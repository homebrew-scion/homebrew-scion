# homebrew-scion

Homebrew tap for the [Scion](https://github.com/GoogleCloudPlatform/scion) multi-agent orchestration platform.

## Installation

```bash
brew tap homebrew-scion/scion
brew install homebrew-scion/scion/scion
```

## Upgrade

```bash
brew update && brew upgrade scion
```

## About

Scion is a multi-agent orchestration platform. For more information, see the main project repository:

https://github.com/GoogleCloudPlatform/scion
